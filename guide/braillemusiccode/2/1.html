2. Notes and Note Values

2.1. Note Names and Values

The notes of one octave of the scale of C are given in Table 2. The eighth note (quaver) consists of only the name of the note occupying dots 1245. Dot 6 is added to form the quarter note (crotchet), dot 3 the half note (minim), and dots 36 the whole note (semibreve). Each sign also represents a smaller value. The number of notes in a measure will usually be a sufficient guide to the reader in determining to which of the two values any note belongs.
