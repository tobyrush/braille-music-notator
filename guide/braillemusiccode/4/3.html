Music Braille Code (2015)

4. Clef Signs

4.3. Staff Line Given with Clef Sign

When it is necessary to show the line on which a clef is placed, as must be shown with the tenor, baritone, and French violin clefs, the last cell of the sign l is preceded by a mark designating the number of the line. The mark is the same sign as the one that represents that number as an octave mark.
