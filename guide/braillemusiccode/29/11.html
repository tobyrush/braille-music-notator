Music Braille Code (2015)

29. Keyboard Instruments

29.11. Piano Pedaling and Repeats

Braille numeral repeats may only be used when the pedaling is exactly the same in the repetition as it is in the original passage. Braille measure and part-measure repeats may be used in conjunction with pedaling indications only when the intention remains perfectly clear. The repeat devices should not be used if there is any doubt.
