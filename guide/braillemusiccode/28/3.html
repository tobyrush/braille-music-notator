Music Braille Code (2015)

28. Instrumental Bar-Over-Bar Format

28.3. Parallel Movement and Sequence Abbreviation

The parallel movement device (see Pars. 18.6–18.6.1 and 33.6) is used wherever it is appropriate. Sequence abbreviation (see Pars. 18.7–18.7.2) is normally reserved for technical studies and other exercise materials that are diatonic in nature. It should only be used where all parts are similarly sequenced.
