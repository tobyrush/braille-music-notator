Music Braille Code (2015)

31. Harps

31.3. Damping of Strings

31.3.1. Damp a Single String or Chord

The damping sign is placed before a single note or chord of rhythmic value that is to be damped. However, if the damping sign is clearly positioned in the print to indicate that the note or chord is to be damped after it is sounded, the modified symbol "*C is brailled immediately following the note.
