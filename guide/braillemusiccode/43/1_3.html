Music Braille Code (2015)

43. Note-for-Note

43.1. Note-for-Note Method

43.1.3. Octave Marks on Subnotes

Subnotes are given octave marks according to the following rules:

  1. When the distance between a subnote and the main note is greater than an octave, the subnote must have an octave mark.
  2. If more than one subnote follows the main note, no octave mark is needed so long as any two adjacent subnotes are less than an octave apart.
  3. If any two adjacent subnotes are an octave or more apart, the second subnote must have its proper octave mark.
  4. If a subnote forms a unison with the main note, it must have its proper octave mark.
  5. If two adjacent subnotes form a unison, the second must have its proper octave mark.
