Music Braille Code (2015)

31. Harps

31.3. Damping of Strings

31.3.6. Delay Damping of a String

Sometimes a line is printed leading from where a particular string is sounded to an encircled cross at a later rhythmic point where that string is to be damped. The sign <C precedes the note at which the leading line begins. The damping sign is brailled where it occurs.
