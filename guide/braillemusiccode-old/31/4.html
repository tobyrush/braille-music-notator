Music Braille Code (2015)

31. Harps

31.4. Special Indication for Fingernail Technique

A crescent-shaped symbol is sometimes shown in print when a string is to be plucked with the fingernail instead of with the flesh of the finger. The braille sign representing this indication, a letter n preceded by a word sign, is placed before the note, and is positioned and treated as any other word-sign expression.
