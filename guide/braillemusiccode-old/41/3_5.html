Music Braille Code (2015)

41. Vertical Score and Bar-by-Bar

41.3. Bar-by-Bar

41.3.5. Measure Rests

When all the parts have a measure of silence, only one rest is written, and for a silence of more than one measure, the procedure shown in Example 41.3.5-1 must be adopted. When the silence is not in all parts, rests are written separately for each part, as shown in Example 41.3.5-2. Where one part rests for more than four measures it is omitted, as shown in Example 41.3.5-3.
