Music Braille Code (2015)

25. Bowed String Instruments

25.2. Interpreting Print Roman Numerals

25.2.2. Positions

The signs for the various positions are placed before the notes and before such initial signs as nuances. The note requires a special octave mark. A line of continuation following a position sign is shown by appending two successive dot 3s (or two successive dots 36 if another line of continuation is already in effect) to the sign. The end of the continuation is shown by inserting the termination sign >' (or >- for a second line of continuation) following the last affected note. The next note requires a special octave mark. The end of a line of continuation is not shown when a fresh position sign is marked.
