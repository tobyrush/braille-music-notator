Music Braille Code (2015)

25. Bowed String Instruments

25.2. Interpreting Print Roman Numerals

Roman numerals may appear above or below the notes to indicate either the number of the string on which to play or to indicate the hand position. Roman numerals are not used for both purposes in the same score. It is necessary to ascertain which function they have in a given score because in neither case will they be transcribed as roman numerals. A transcriber who is not knowledgeable about string instruments must seek advice from a performer or a reference.
