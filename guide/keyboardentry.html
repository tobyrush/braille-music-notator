Keyboard Entry

Braille Music Notator features different keyboard input maps for different purposes. A description of each keyboard input map is included below in three formats: a text file suitable for screen readers, a BRF file suitable for embossing, and a Braille Music Notator file which can be edited if desired.

To change keyboard input layouts, select "Controls" in the Options tab, or use the Control-` keyboard shortcut.
