Change Log

Version 0.9.4 (beta) · {date}

Braille Music Notator:

  • Redesigned file format to XML-based open format
  • Added ability to add symbols by character name (Command/Control-\)
  • Added complete set of braille music symbols as defined in 2015 BANA Music Braille Code
  • Replaced "IPA" controls layout with "Full" controls layout
  • Changed some terminology to align with 2015 BANA Music Braille Code
  • Fixed text contraction symbols to prevent overlap
  • Changed coloring of some symbols
  • Fixed bug which caused score selections to grow erratically
  • Fixed bug which caused clicks to be ignored when dialog boxes were open
  • Fixed bug which caused the canvas to scroll after choosing select all and delete

Braille Music Viewer:

  • Rebuilt from the ground up to support new file format
  • Added feature to convert <span class="braille">...</span> in body text to Unicode braille

Version 0.9.3 (beta) · 17 September 2019

  • Added MIDI Note Entry (available in Chrome and Firefox only)
  • Added capability to import MusicXML files via Dr. Toshiyuki Gotoh's BrailleMUSE translation engine
  • Added more musical symbols:
    • additional repeat structures
    • microtonal accidentals (thanks to Dr. Sevan Nart)
    • ...and others
  • Expended text keyboard controls
  • Added modular control system
  • Improved handling of large scores
  • Changed spacebar to insert space, changed keyboard commands for changing control pages to tab and shift-tab
  • Added automatic end-of-line word wrap with octave symbol placement
  • Improved parsing of BRF files
  • Created open source GitHub repository
  • Fixed assorted bugs

Version 0.9.2 (beta) · 4 February 2018

  • Changed File and Open dialogs to be modeless
  • Added support for refreshable braille displays

Version 0.9.1 (beta) · 15 June 2017

  • Fixed error that prevented screenreader from working properly
  • Braille Music Viewer: added screenreader compatibility:
    • added dynamic creation of tabular presentation of score
    • added Control/Command-U keyboard command to switch between presentation styles
  • Added chord symbol characters:
    • plus
    • diminished
    • half-diminished
    • major
  • Lightened borders of key caps for text keys
  • Cleaned up code site-wide

Version 0.9.0 (beta) · 18 February 2016

  • Added parsing of imported files
  • Added new symbols:

    • text dynamics (cresc., decresc., dim.)

    • note value signs
    • capitalize word symbol
    • multi-measure rests

    • trill
    • grace notes
    • in-accord symbols

  • improved contextual help

  • fixed error that caused dot-3 braille symbol to sometimes be hidden
  • fixed icons in online help for tuplets
  • fixed error that prevented notation and control areas from resizing when the window is resized
  • Braille Music Viewer: added new symbols to sync with Braille Music Notator

  • updated project to beta status!

Version 0.8.8 (alpha)

  • Added music and text prefixes, crescendos and diminuendos, articulations, measure repeats and slurs
  • Changed symbol to better delineate ties and slurs

Version 0.8.6 (alpha)

  • Split into separate production and development versions
  • Changed Time Signature icons to be more clear
  • Changed Paste function to Paste as Text when the text keyboard is selected

  • Added page numbers in right margin when "Show Page Boundaries" is selected
  • Changed title to "Braille Music Notator" and began version numbering
  • Added automatic autosave to local storage (score state and previous 5 undos will be restored if utility is closed and reopened)
  • Added Rotate Character (Ctrl-R) which cycles through various meanings of the selected braille character(s)

  • Fixed an issue where various selection-dependent functions (like copy/paste, rotate character or convert to text) would behave unpredictably
